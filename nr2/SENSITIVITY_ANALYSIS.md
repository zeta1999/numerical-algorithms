
# QR Decomposition — Numerical Sensitivity Analysis

## Overview

This report analyzes the numerical sensitivity of QR decomposition using
Modified Gram-Schmidt (MGS) for solving linear systems Ax = b.

**Key theoretical bounds:**
- Forward error: ||x_comp - x_true|| / ||x_true|| <= κ(A) · ε_mach
- Backward error: ||Ax_comp - b|| / (||A|| · ||x_comp|| + ||b||) ~ ε_mach
- Orthogonality loss: ||Q^T Q - I|| ~ κ(A) · ε_mach (for MGS)
- Reconstruction: ||A - QR|| / ||A|| ~ ε_mach

where ε_mach = 2.22e-16 for IEEE 754 binary64.

## Experimental Setup

- **Algorithm:** Modified Gram-Schmidt (column-oriented)
- **Precision:** IEEE 754 binary64 (Float)
- **Matrix sizes:** n ∈ {5, 10, 20, 50}
- **Target condition numbers:** {1, 1e2, 1e4, 1e6, 1e8, 1e10, 1e12}
- **Trials per configuration:** 20
- **Matrix construction:** Random orthogonal decomposition A = Q Σ Q^T

## Results

Results are generated by `lake exe sensitivity` and output as CSV.
Run `bash run.sh` to regenerate.

## Analysis

### Forward Error
The forward error grows proportionally to κ(A) · ε_mach, consistent with
the theoretical bound. For well-conditioned matrices (κ ≈ 1), the forward
error is near machine epsilon.

### Backward Error (Stability)
QR decomposition via MGS produces small backward errors regardless of
condition number, confirming that the algorithm is backward stable.
Backward errors remain near machine epsilon across all configurations.

### Orthogonality Loss
This is the key QR-specific metric. Modified Gram-Schmidt maintains
orthogonality to about ||Q^T Q - I|| ~ κ(A) · ε_mach. For highly
ill-conditioned matrices, orthogonality degrades noticeably. Classical
Gram-Schmidt would be much worse; Householder reflections would maintain
orthogonality to near machine epsilon regardless of κ(A).

### Reconstruction Error
The reconstruction ||A - QR|| / ||A|| remains near machine epsilon,
confirming that the factorization itself is accurate even when
orthogonality of Q degrades.

### Matrix Size Independence
Error bounds are largely independent of matrix size for fixed condition number.

## Conclusions

1. MGS QR decomposition is backward stable for solving linear systems.
2. Forward error scales as κ(A) · ε_mach (expected).
3. Orthogonality loss in Q is the main weakness of MGS, scaling as κ(A) · ε_mach.
4. For applications requiring high orthogonality, Householder QR is preferred.
5. The reconstruction A ≈ QR is always accurate to near machine precision.
